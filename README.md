# Event Room Organizer

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-blue?logo=github)](https://salva.codes/event-room-organizer/)
[![CI/CD pipeline](https://github.com/salvacodes/event-room-organizer/actions/workflows/workflow.yml/badge.svg?branch=main)](https://github.com/salvacodes/event-room-organizer/actions/workflows/workflow.yml)

An interactive event room and bed planning tool. Drag-and-drop participants into beds, upload rooms and registrations via CSV, and export custom print-ready room allocations to PDF.

## Getting started

### Prerequisites

* Node 24.x 

### Running the app

1. Install dependencies: `npm i`
2. Run the app locally: `npm run dev`

### Verifications

* Type checking: `npm run check:type`
* Formatting and Linting: `npm run check:fix` (or `npm run check` to just verify without fixing)
* Unit tests: `npm run test`
